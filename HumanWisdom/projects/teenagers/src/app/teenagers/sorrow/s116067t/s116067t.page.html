If we are human, we are going to lose people and things we value and this is going to be painful. We can be quickly overwhelmed by the number of emotions we are faced with, over which we have no control, and which can create an unbearable sadness.How can we use our wisdom to deal with this in a fresh way? Welcome to this session where we are going to apply everything we have learned so far to explore how we can meet loss with wisdom. This understanding can help us prepare for loss when it happens - and deal with it more effectively. It's like learning to swim, so when the challenge arrives we can respond with wisdom.
