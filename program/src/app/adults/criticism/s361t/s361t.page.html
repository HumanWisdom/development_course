We have explored how hurtful criticism can be and how it can invite an angry backlash. So its worth asking yourself a few questions before you criticize another:

1. Is this for my benefit or for the other person's benefit?

2. How is it going to be received?

3. Is this the best time to say it?

4. What would happen if I said nothing?

5. Can I find something to praise instead? Much more effective at bringing change.
